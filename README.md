# TEKO-BOARD

TEKO事業の人員配置・現場管理システム

## 概要

TEKO（軽作業人材派遣）の運営管理を目的としたWebアプリケーション。
人員配置のタイムチャート表示、現場管理、連絡先検索を中心に、効率的な配車・人員管理を支援する。

## 主な機能（Phase 1）

- **タイムチャート**: 誰がどの現場にいるかを一目で把握
- **現場一覧**: 進行中の現場をリスト表示
- **スケジュール登録**: 予定の追加・編集
- **連絡先検索**: 会社・職人の連絡先管理

## 技術スタック

| 項目 | 選定 |
|------|------|
| 開発方法 | Cursor + Claude（AI駆動開発） |
| フレームワーク | React 18 + TypeScript |
| スタイリング | Tailwind CSS |
| ビルドツール | Vite |
| データ管理 | ローカルステート → Supabase（予定） |
| ホスティング | Vercel（予定） |

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
