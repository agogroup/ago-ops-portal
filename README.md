# AGO Ops Portal

AGOグループの現場運営支援ツール

## 概要

内装現場の運営効率化を目的としたWebアプリケーション。
TEKO（ゴミ回収・雑工事部隊）の配車管理を中心に、現場関係者の連携を支援する。

## 主な機能（Phase 1）

- **タイムチャート**: 誰がどの現場にいるかを一目で把握
- **現場一覧**: 進行中の現場をリスト表示
- **スケジュール登録**: 予定の追加・編集
- **連絡先検索**: 会社・職人の連絡先管理

## 利用者

- 染谷雅人（TEKO現場管理）
- 菅優伍（TEKOデスク管理）
- 堀川寿人/トラ（現場管理）
- 梅内光（積算・工程準備）

## 技術スタック

| 項目 | 選定 |
|------|------|
| 開発方法 | Cursor + Claude（AI駆動開発） |
| フレームワーク | React 18 + TypeScript |
| スタイリング | Tailwind CSS |
| ビルドツール | Vite |
| データ管理 | ローカルステート → Supabase（予定） |
| ホスティング | Vercel（予定） |

## 開発

```bash
# 依存関係インストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
