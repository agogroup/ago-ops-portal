# AGO Ops Portal

AGOグループの現場運営支援ツール

## 概要

内装現場の運営効率化を目的としたWebアプリケーション。
TEKO（ゴミ回収・雑工事部隊）の配車管理を中心に、現場関係者の連携を支援する。

## 主な機能（予定）

- TEKO配車管理
- 現場ステータス一覧
- 職人・業者連絡先管理

## 利用者

- 染谷雅人（TEKO現場管理）
- 菅優伍（TEKOデスク管理）
- 堀川寿人/トラ（現場管理）
- 梅内光（積算・工程準備）
