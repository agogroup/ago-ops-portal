# 品質チェック

変更したコードの品質をチェックしてください。

## チェック項目

1. **ビルドエラー**: `npm run build` を実行してエラーがないか確認
2. **型エラー**: TypeScriptの型定義が正しいか
3. **セキュリティ**: 機密情報が含まれていないか（.claude/rules/security.md 参照）
4. **コード品質**:
   - 不要なconsole.logが残っていないか
   - 未使用のimportがないか
   - ハードコードされた値がないか

## 出力形式

```
## 品質チェック結果

### ビルド: ✅ OK / ❌ NG
[詳細]

### 型チェック: ✅ OK / ❌ NG
[詳細]

### セキュリティ: ✅ OK / ⚠️ 要確認
[詳細]

### コード品質: ✅ OK / ⚠️ 改善推奨
[詳細]
```
